# **Project Title**
 
**基于Vue2.0的GPTweb客户端
 
## Getting Started  

* [**NODE=14**](golang_install_guide)
  
## Major Function
--客户端  

* **登录**
* **注册赠送10次对话**
* **对话记录**
* **公告查看**
* **个人信息展示（剩余次数，身份，昵称）**
* **产品查询购买（支持支付宝，微信，QQ钱包）**
* **订单查询**
* **注册验证码（正在开发）**
* **邀请下单返现（正在开发）**  
* 
--管理端  

* **首页（数据统计）**
* **支付配置**
* **对KEY配置**
* **用户管理**
* **订单管理**
* **公告管理**
* **产品管理**


 
## Installing
 
**1.安装node**  

**2.yum install**  

**3.npm run dev**
